---
title: Installation
description: Install Minions Jobs packages
---

## TypeScript / Node.js

```bash
npm install @minions-jobs/sdk minions-sdk
```

## Python

```bash
pip install minions-jobs
```

## CLI

```bash
npm install -g @minions-jobs/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
