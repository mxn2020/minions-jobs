# Minions Jobs

Canonical schemas for job postings and extracted signals across freelance platforms

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-jobs/sdk`) — Core library for Node.js
- **Python SDK** (`minions-jobs`) — Python bindings
- **CLI Tool** (`@minions-jobs/cli`) — Command-line interface
- **Web App** — Interactive playground at [jobs.minions.wtf](https://jobs.minions.wtf)
